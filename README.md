Automated [rollup](https://github.com/rollup/rollup) build based on
[babel](https://github.com/rollup/rollup-plugin-babel).

## Table of Contents

* [Install](#install)

## Install

```
yarn add --dev bup
```

## Usage

```bash
bup . # compiles pkg.entry to dist/{name}.{format}.js
```
